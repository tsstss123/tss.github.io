CF 615D Multipliers 数学

发表于 2016-01-09   |  

题意

给你一个$m\left( 1\le m\le 200\ 000 \right)$,有$m$个素因子构成$n$.求n的所有因数的乘积对1e9+7取模.

分析

考虑把$n$分解成$a\ast b$,那么对应每一个$a$有唯一的$b$与之对应.假定有$k$个$a$,所求答案即为${ n }^{ \frac { k }{ 2 } }$ 考虑如何求$k$?由于给出的是所有的素因子,所以可以很容易的得到$n=\sum { { { p }_{ i } }^{ { d }_{ i } } }$其中$p_{ i }$为其中一个素因子,${ d }_{ i }$为次数.所以$k=\prod { \left( { d }_{ i }+1 \right) }$

算法1

一个比较朴素的想法就是先求出$n$然后再考虑求$\frac { k }{ 2 }$次幂.这里又有两种选择:

  • 无脑快速幂(我的做法).对$k$是否能整除$2$进行分类对,若无法整除则对第一个$d$除$2$再给当前ans乘上$\sqrt { n }$即$\prod { { { p }_{ i } }^{ \frac { { d }_{ i } }{ 2 } } }$(同样使用快速幂),对余下的每一个$d$都进行快速幂计算.
  • 费马小定理.由于整体对1e9+7取模,按照费马小定理,指数对1e9+6取模即可.处理除法使用如下公式:$\frac { a }{ b } \left( mod \ \ m \right) =\frac { a\ mod\ \left( m\ast b \right) }{ b } \left( mod \ \ m \right)$

算法2

不求$n$,将$k$展开至各个$p$的指数上.答案为$\sum { { { p }_{ i } }^{ \frac { { { d }_{ i } \left ( { d }_{ 1 }+1 \right) \left( { d }_{ 2 }+1 \right) \cdots \left( { d }_{ n }+1 \right) } }{ 2 } } }$.从指数中提出$\frac { { d }_{ i }\ast \left( { d }_{ i }+1 \right) }{ 2 } =1+2+3+\cdots +{ d }_{ i }$.处理一下$\prod { { d }_{ i } + 1}$的前缀和后缀和,并用费马小定理处理即可.

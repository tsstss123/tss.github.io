Uva 3767 Dynamic len(set(a[L:R])) 树套树STL

发表于 2015-12-06   |  

题意

给你一个5w的序列,5w的长度.序列上每个点都有一个范围在1-100w的数值.每个询问给出区间内有多少种值.

转化

将原序列转化成一个last序列.新序列中每个位置上的值改成原序列中与这个位置的值相同的前一个值的位置.

样例
1 2 1 3 2 1 4
变成
0 0 1 0 2 3 0
然后询问的答案就可以变成在新序列中区间内的长度len,减去区间内值大于区间左端点位置的值的数量
即询问[2,5]变成(5-2+1)-在新序列的[2,5]中>=2的值得数量

维护

先观察这个新序列的一些性质:

  • 在任意的区间内(或者说整个序列内部)不会出现除了0以外的数字重复,而我们的询问的值永远大于0
  • [1-x]内不会出现>=x的值

根据性质1,我们可以直接忽略掉新序列中值为0的位置
根据性质2,我们可以将[L,R]的询问变成[1,R].把线段树换成树状数组减小常数(这个性质后来才想起来…程序里还是线段树)

最重要的是…根据性质1…我们可以用STL里的平衡树来搞…

STL大法好!
